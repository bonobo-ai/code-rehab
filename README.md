# Code Rehab

Landing page for Code Rehab — a service helping non-technical founders fix their AI-built apps.

## What is this?

A single-page site targeting founders who used vibe coding tools (Lovable, Cursor, Bolt) to build apps that look great but don't actually work. The site explains common problems with AI-generated code and offers professional debugging and development services.

## Tech Stack

- **Astro** — Static site generator
- **Vanilla CSS** — Custom design system with teal color palette
- **Schema.org** — Structured data for SEO and AI search

## Development

```bash
npm install
npm run dev
```

## Deployment

Hosted on Cloudflare Pages at [coderehab.me](https://coderehab.me)
